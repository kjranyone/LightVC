## 1. ウィンドウ全体構成

```
┌──────────────────────────────────────────────────┐
│  [ロゴ]  [Offline] [Realtime] [Voices]       [=] │  ← トップバー
├──────────────────────────────────────────────────┤
│                                                  │
│              タブ別コンテンツ                      │  ← 中央エリア
│           （縦スクロール可能）                      │
│                                                  │
├──────────────────────────────────────────────────┤
│  ● Out [█████░░░]  Converter loaded              │  ← ステータスバー
└──────────────────────────────────────────────────┘
```

| 領域 | 説明 |
|------|------|
| **トップバー** | ロゴ、タブ切替ボタン、設定ボタン(=) |
| **中央エリア** | 選択中タブのコンテンツ（縦スクロール可能） |
| **ステータスバー** | 出力メーター＋アプリ状態メッセージ |

## 2. トップバー・ステータスバー

### タブ切替ボタン

| 英語ラベル | 日本語 | 説明 |
|-----------|--------|------|
| `Offline` | オフライン変換 | WAV ファイル → WAV ファイルのバッチ変換 |
| `Realtime` | リアルタイム変換 | マイク入力をリアルタイム変換してスピーカー出力 |
| `Voices` | ボイスカタログ | ゼロショット変換用の参照音声を管理 |

> 選択中のタブは**ピンク色**にハイライトされます。

### 設定ボタン

| 英語ラベル | 日本語 | 説明 |
|-----------|--------|------|
| `=` | 設定 | クリックすると Audio Devices ウィンドウが開く |

### ステータスバー（画面下部）

```
● Out [██████████░░░░░░] -18 dB    Converter loaded
```

| 表示 | 意味 |
|------|------|
| `Out` ラベル + メーター | リアルタイム変換中の出力音量レベル |
| `●`（緑/黄/赤のドット） | アプリ状態（緑=正常、黄=警告、赤=エラー） |
| 右側のテキスト | 現在のアプリ状態メッセージ（[メッセージ一覧](#7-ステータスエラーメッセージ一覧)参照） |

## 3. Offline タブ（オフライン変換）

WAV ファイルを別の声に変換します。

### 画面構成

```
┌─ Source ────────────────────────────────────┐
│ 🎤 [source.wav___________] [▶ Play][Browse] │
└─────────────────────────────────────────────┘

┌─ Reference ─────────────────────────────────┐
│ 🔊 [target.wav__________] [▶ Play][Browse] │
└─────────────────────────────────────────────┘

  Or pick from Voice Catalog:
  [Venus (warm)] [Mars (bright)] [Lyra (soft)]

┌─ Prosody ───────────────────────────────────┐
│ Mode:   [Imitate target ▾]                   │
│ Blend:  [──────●──────] source <-> target    │
└─────────────────────────────────────────────┘

┌─ Conversion Strength ───────────────────────┐
│ [──────●──────────] velocity scale  default  │
└─────────────────────────────────────────────┘

         ✦  Convert

┌─ Output ────────────────────────────────────┐  ← 変換後のみ表示
│ 44100 samples (1.0s)                         │
│ [▶ Play Output] [💾 Save As...]              │
└─────────────────────────────────────────────┘
```

### Source カード（変換元音声）

| 英語ラベル | 日本語 | 説明 |
|-----------|--------|------|
| `Source` 🎤 | 変換元 | 変換したい音声ファイル（自分の声 etc.） |
| `source audio file` | （グレーテキスト） | パス未入力時のヒント |
| `Play` / `Stop` | 再生 / 停止 | 変換元音声を試聴（再生中は Stop に切替） |
| `Browse` | 参照 | ファイル選択ダイアログを開く |

### Reference カード（変換先の声）

| 英語ラベル | 日本語 | 説明 |
|-----------|--------|------|
| `Reference` 🔊 | 参照音声 | この声に変換したいターゲット音声（5〜30秒推奨） |
| `target voice reference` | （グレーテキスト） | パス未入力時のヒント |
| `Play` / `Stop` | 再生 / 停止 | 参照音声を試聴 |
| `Browse` | 参照 | ファイル選択ダイアログを開く |

### ボイスカタログ クイック選択

| 英語ラベル | 日本語 | 説明 |
|-----------|--------|------|
| `Or pick from Voice Catalog` | または カタログから選択 | Voices タブで登録済みのボイス一覧（ボタンで選択） |

### Prosody（抑揚制御）

| 英語ラベル | 日本語 | 説明 |
|-----------|--------|------|
| `Prosody` | 抑揚 | 音声の抑揚（エネルギーエンベロープ）の制御モード |
| `Mode` | モード | 下記4モードから選択 |
| `Blend` | ブレンド | ソースとターゲットの抑揚ミックス比率（Blend モード時のみ有効） |
| `source <-> target` | 元の声 ↔ ターゲット | ブレンドスライダーの両端ラベル |

**Mode 選択肢:**

| 英語 | 日本語 | 動作 |
|------|--------|------|
| `Imitate target` | ターゲットを模倣 | 変換モデルが推定するターゲットの抑揚を適用（デフォルト） |
| `Preserve source` | 元の声を保持 | 元話者の抑揚・テンポ・強弱を完全維持 |
| `Blend` | ブレンド | ソースとターゲットの抑揚を指定比率でミックス |
| `Flatten (privacy)` | 平坦化（プライバシー） | 抑揚をほぼ平坦化（匿名化音声） |

### Conversion Strength（変換強度）

| 英語ラベル | 日本語 | 説明 |
|-----------|--------|------|
| `Conversion Strength` | 変換強度 | 変換の強さを調整するスライダー |
| `velocity scale` | 速度スケール | フローマッチング推論の速度ベクトルへの乗数 |

**スライダー補助ラベル（値に応じて動的表示）:**

| 英語 | 日本語 | 範囲 |
|------|--------|------|
| `<-- mild` | ← マイルド | 0.9 未満（元話者の声質が残る） |
| `default` | 標準 | 0.9〜1.1 |
| `strong -->` | 強変換 → | 1.1 超（ターゲットに強く引き寄せ、歪みリスクあり） |

### 変換実行・出力

| 英語ラベル | 日本語 | 説明 |
|-----------|--------|------|
| `Convert` ✦ | 変換 | 変換を実行（CTA ボタン） |
| `Converting...` | 変換中... | 変換処理中の表示（スピナー付き） |
| `Output` | 出力 | 変換結果セクション（変換完了後に表示） |
| `{N} samples ({X.X}s)` | {N} サンプル（{X.X}秒） | 出力音声の長さ |
| `Play Output` / `Stop` | 出力を再生 / 停止 | 変換結果を試聴 |
| `Save As...` | 名前を付けて保存 | 変換結果を WAV ファイルに保存 |

### 警告メッセージ

| 英語 | 日本語 | 表示条件 |
|------|--------|---------|
| `No converter loaded. Set model in Realtime tab.` | コンバーターがロードされていません。Realtime タブでモデルを設定してください。 | コンバーター未ロード時（赤色表示） |

## 4. Realtime タブ（リアルタイム変換）

マイク入力をリアルタイムに変換してスピーカーに出力します。

### 画面構成

```
┌─ Model ──────────────────────┐  ┌─ Status ───────────────────┐
│ Drop converter.safetensors    │  │ CONVERTING                  │
│ or click Browse below         │  │ > Venus (warm)              │
│ [Browse]                      │  │                             │
│                               │  │  46 ms        0.34 RTF      │
│ [*] Config                    │  │  in:0  out:2                │
│  latent [512] hidden [768]    │  └─────────────────────────────┘
│  ...                          │
│ [Load]                        │  ┌─ Input ────────────────────┐
│                               │  │ -15 dB                     │
│ no model -> BYPASS            │  │ [████████████░░░░░]        │
└───────────────────────────────┘  └────────────────────────────┘

  [Bypass]  [Strict] [Balanced] [Quality]        [■ Stop]

┌─ Prosody ───────────────────────────────────┐
│ Mode: [Blend ▾]                              │
│ blend [──────●──────]                        │
│ velocity [─────●─────]                       │
└─────────────────────────────────────────────┘
```

### Model カード（モデルロード）

| 英語ラベル | 日本語 | 説明 |
|-----------|--------|------|
| `Model` | モデル | 変換モデル（コンバーター）のロードセクション |
| `Drop converter.safetensors here` | converter.safetensors をここにドロップ | ファイルをドラッグ&ドロップで指定（ドロップゾーン） |
| `or click Browse below` | または下の Browse をクリック | ドロップゾーンの補助ヒント |
| `Browse` | 参照 | ファイル選択ダイアログ |
| `[*] Config` | 設定 | モデル設定（アーキテクチャパラメータ）の折りたたみ |
| `Load` | ロード | 指定したコンバーターを読み込む |
| `no model -> BYPASS` | モデル未ロード → バイパス | コンバーター未ロード時の警告（赤色） |
| `ready` | 準備完了 | コンバーターロード済みの状態表示 |

#### Config 詳細パラメータ

`[*] Config` を展開すると以下のフォームが表示されます:

| 英語ラベル | 日本語 | 説明 |
|-----------|--------|------|
| `latent` | 潜在次元 | DAC潜在表現の次元数（例: 512） |
| `hidden` | 隠れ層次元 | 変換器の隠れ層サイズ（例: 768） |
| `blocks` | ブロック数 | Transformerブロック数（例: 6） |
| `spk` | 話者埋め込み | 話者エンベディング次元（例: 256） |
| `timbre` | 音色次元 | 音色エンベディング次元 |
| `heads` | アテンションヘッド数 | Multi-head Attention のヘッド数 |
| `bneck` | ボトルネック | 情報ボトルネックの次元 |
| `time` | 時間方向 | 時間方向の畳み込みカーネルサイズ |
| `depth` | 深さ | レイヤー深さ |
| `timbre`（チェックボックス） | 音色条件付け | 音色エンベディングを使用するか |

### Status カード（状態表示）

| 英語ラベル | 日本語 | 説明 |
|-----------|--------|------|
| `Status` | 状態 | リアルタイム動作状態の表示 |
| `CONVERTING` | 変換中 | リアルタイム変換動作中 |
| `BYPASS` | バイパス | 変換せず入力をそのまま出力中 |
| `STOPPED` | 停止中 | リアルタイム処理が停止中 |
| `> {model_name}` | > {モデル名} | ロード中のコンバーター名 |
| `> {voice_name}` | > {ボイス名} | 選択中の参照ボイス名 |
| `no voice` | ボイス未選択 | 参照音声が選択されていない |
| `ms` | ミリ秒 | エンドツーエンドレイテンシの単位 |
| `RTF` | リアルタイムファクター | 処理時間/音声時間（1.0未満=リアルタイム可能） |
| `in:{N}` | 入力:{N} | capture overrun（キャプチャ溢れ）の回数 |
| `out:{N}` | 出力:{N} | playback underrun（再生途切れ）の回数 |
| `! auto {mode}` | 自動降格 {モード} | underrun 連続発生による自動レイテンシ降格中 |

### Input カード（入力レベルメーター）

| 英語ラベル | 日本語 | 説明 |
|-----------|--------|------|
| `Input` | 入力 | マイク入力レベルメーター |
| `dB` | デシベル | 音量レベル単位 |
| `{db:+.0}` | {数値}dB | リアルタイム音量表示（例: -15 dB） |

### 操作バー（Operation Bar）

| 英語ラベル | 日本語 | 説明 |
|-----------|--------|------|
| `Bypass` / `BYPASS` | バイパス | 変換をバイパス（入力をそのまま出力）。ON時は大文字+ハイライト |
| `Strict` | 厳格（最低遅延） | レイテンシ 0ms（ルックアヘッドなし） |
| `Balanced` | バランス | レイテンシ ~46ms（推奨） |
| `Quality` | 高品質（最大遅延） | レイテンシ ~93ms（最高品質） |
| `Start` | 開始 | リアルタイム変換を開始 |
| `Stop` ■ | 停止 | リアルタイム変換を停止 |

> **注意**: コンバーター未ロード時は `Start` ボタンが bypass モードで動作し、`Strict`/`Balanced`/`Quality` ピルは非表示になります。

### Prosody・Velocity（リアルタイム版）

Offline タブと同等の機能ですが、ラベルが短縮形です:

| 英語ラベル | 日本語 | 説明 |
|-----------|--------|------|
| `Prosody` | 抑揚 | 抑揚制御セクション |
| `Mode` | モード | 抑揚モード選択 |
| `Imitate` | 模倣 | ターゲットの抑揚を模倣（短縮形） |
| `Preserve` | 保持 | 元の声の抑揚を保持（短縮形） |
| `Blend` | ブレンド | 抑揚をミックス（短縮形） |
| `Flatten` | 平坦化 | 抑揚を平坦化（短縮形） |
| `blend` | ブレンド | ブレンド比率スライダー |
| `velocity` | 強度 | 変換強度スライダー |

### Audio Devices ウィンドウ

`=` ボタンから開きます:

| 英語ラベル | 日本語 | 説明 |
|-----------|--------|------|
| `Audio Devices` | オーディオデバイス | 入出力デバイス選択ウィンドウ |
| `Input` | 入力 | マイク/入力デバイス一覧 |
| `Output` | 出力 | スピーカー/出力デバイス一覧 |
| `(default)` | （デフォルト） | システムのデフォルトデバイス（未選択時） |
| `{name} ({sr}Hz, {ch}ch)` | {デバイス名}（{サンプリングレート}Hz、{チャンネル数}ch） | 各デバイスの詳細情報 |

## 5. Voices タブ（ボイスカタログ）

ゼロショット変換用の参照音声を管理します。

### 画面構成

```
  Register reference audio for zero-shot voice conversion

┌─ Add Voice ─────────────────────────────────┐
│ Name [_____________] [Browse] [Add]          │
└─────────────────────────────────────────────┘

  3 voices

┌──────────────────────────────────────────────┐
│ ★ Venus (warm)              [Use][Play][🗑]   │  ← 選択中
│   /path/to/venus.wav                          │
└──────────────────────────────────────────────┘
┌──────────────────────────────────────────────┐
│ 2  Mars (bright)            [Use][Play][🗑]   │
│   /path/to/mars.wav                           │
└──────────────────────────────────────────────┘

  ▶ Import / Export
```

### 英語ラベル

| 英語ラベル | 日本語 | 説明 |
|-----------|--------|------|
| `Register reference audio for zero-shot voice conversion` | ゼロショット声質変換用の参照音声を登録します | ページ説明文 |
| `Add Voice` | ボイス追加 | 新しい参照音声を登録するセクション |
| `Name` | 名前 | ボイスの表示名 |
| `Browse` | 参照 | 音声ファイル（WAV）を選択 |
| `Add` | 追加 | 入力した名前とファイルでボイスを登録 |
| `{N} voices` | {N}ボイス | 登録済みボイス数 |
| `★` / `*` | 選択中マーク | 選択中のボイス（ピンク色の星） |
| `{N}` | {連番} | ボイスの行番号（1始まり） |
| `Use` | 使用 | このボイスを参照音声として選択（Realtime タブに反映） |
| `[v] Selected` | [v] 選択中 | 既に選択されている状態のボタン表示 |
| `Play` / `Stop` | 再生 / 停止 | 参照音声を試聴 |
| `Remove` 🗑 | 削除 | ボイスをカタログから削除 |
| `No voices registered yet.` | まだボイスが登録されていません | 空状態メッセージ |
| `Import / Export` | インポート / エクスポート | JSON形式でボイスリストを入出力（折りたたみ） |
| `Import` | インポート | JSONファイルからボイスリストを読み込み |
| `Export` | エクスポート | ボイスリストをJSONファイルに書き出し |

## 6. B1 Adapter セクション

Realtime タブの下部に折りたたみ表示される、UTTE（Universal Target-Timbred Encoder）クロスアテンションアダプターの制御セクションです。

```
▼ B1 Adapter (UTTE)
  Adapter:    [________________________]
  Quantizer:  [________________________]
  Timbre:     [________________________]
  [Load B1 Adapter]
  Tau:    [─────●──────] 1.0
  Wet/Dry: [────●───────] 0.3
  ● B1 adapter active
```

| 英語ラベル | 日本語 | 説明 |
|-----------|--------|------|
| `B1 Adapter (UTTE)` | B1 アダプター（UTTE） | UTTEクロスアテンションアダプターのセクション見出し |
| `Adapter:` | アダプター | アダプター重みファイル（.safetensors）のパス |
| `Quantizer:` | 量子化器 | 量子化器重みファイルのパス |
| `Timbre:` | 音色 | ターゲット音色ファイルのパス |
| `Load B1 Adapter` | B1 アダプターをロード | 指定したファイルでB1パイプラインを構築 |
| `Tau:` | τ（タウ） | クロスアテンションの温度パラメータ（0.1〜10.0） |
| `Wet/Dry:` | ウェット/ドライ | 変換音（wet）と原音（dry）のミックス比率（0.0=完全ドライ、1.0=完全ウェット） |
| `● B1 adapter active` | ● B1 アダプター動作中 | B1パイプラインがアクティブな状態（緑色表示） |

## 7. ステータス・エラーメッセージ一覧

### アプリ状態メッセージ（ステータスバー右側に表示）

| 英語メッセージ | 日本語 | 意味 |
|---------------|--------|------|
| `Ready` | 準備完了 | 起動直後の初期状態 |
| `Demo mode` | デモモード | `--demo` フラグで起動中（モックデータ使用） |
| `Converter loaded` | コンバーターをロードしました | コンバーター重みの読み込みに成功 |
| `B1 adapter loaded` | B1 アダプターをロードしました | B1パイプラインの構築に成功 |
| `Audio device disconnected` | オーディオデバイスが切断されました | デバイス切断（再選択が必要） |
| `Converting...` | 変換中... | オフライン変換処理中 |
| `Converted: {N} samples` | 変換完了: {N} サンプル | オフライン変換成功 |
| `Added voice: {name}` | ボイスを追加しました: {name} | ボイスカタログへの追加成功 |
| `Removed voice: {name}` | ボイスを削除しました: {name} | ボイスカタログからの削除成功 |
| `Imported {N} voices` | {N} ボイスをインポートしました | JSON からのインポート成功 |

### エラーメッセージ（赤色表示）

| 英語メッセージ | 日本語 | 原因・対処 |
|---------------|--------|-----------|
| `Load failed: {e}` | ロード失敗: {詳細} | コンバーター重みの読み込みエラー（ファイル破損、パス間違い等） |
| `B1 load failed: {e}` | B1 ロード失敗: {詳細} | B1アダプターの構築エラー |
| `Conversion failed: {e}` | 変換失敗: {詳細} | 変換処理中のエラー |
| `Pipeline not initialized` | パイプラインが初期化されていません | モデル未ロード状態での変換試行 |
| `Error` | エラー | 汎用エラー（詳細は上記メッセージに付随） |

### コンソールログ（ターミナルに表示）

UIには出ませんが、ターミナル/コマンドプロンプトに表示されるログメッセージ:

| 英語ログ | 日本語 | 意味 |
|---------|--------|------|
| `Device selection failed, falling back to default` | デバイス選択失敗、デフォルトにフォールバック | 指定デバイスが利用不可 |
| `Audio: {e}` | オーディオ: {詳細} | オーディオシステムエラー |
| `Audio device disconnected — stopping pipeline` | オーディオデバイス切断 — パイプラインを停止します | デバイス切断検知 |
| `resample up: {e}` | アップサンプリング: {詳細} | リサンプリングエラー |
| `VC: {e}` | VC: {詳細} | 変換処理エラー |
| `resample down: {e}` | ダウンサンプリング: {詳細} | リサンプリングエラー |
| `Auto-degraded to {mode} due to underruns` | アンダーラン発生のため {モード} に自動降格しました | underrun連続による品質自動調整 |

## 用語集

| 英語 | 読み | 日本語 |
|------|------|--------|
| Converter | コンバーター | 変換器（声質変換モデル） |
| DAC | ディーエーシー | Descript Audio Codec（音声圧縮符号化器） |
| Reference | レファレンス | 参照音声（変換先の声） |
| Source | ソース | 変換元音声 |
| Zero-shot | ゼロショット | 事前学習不要の話者適応 |
| Prosody | プロソディ | 抑揚（声の上げ下げ、リズム、強弱） |
| Latency | レイテンシ | 遅延時間 |
| Lookahead | ルックアヘッド | 先読み（未来の音声フレームを先に処理すること） |
| Bypass | バイパス | 変換を通さず入力をそのまま出力 |
| RTF | アールティーエフ | Real-Time Factor（実時間比 = 処理時間 / 音声時間） |
| Underrun | アンダーラン | 再生バッファの空き（音切れ） |
| Overrun | オーバーラン | 録音バッファの溢れ（音抜け） |
| xrun | エックスラン | Overrun/Underrun の総称 |
| Wet/Dry | ウェット/ドライ | 変換音/原音のミックス比率 |
| Timbre | タンブル | 音色 |
| UTTE | ユーティーアールティーイー | Universal Target-Timbred Encoder |
| Velocity | ベロシティ | 変換速度（フローマッチングの速度ベクトルスケール） |
| Flow Matching | フローマッチング | 確率流マッチング（LightVCの推論手法） |
| Safetensors | セーフテンソル | 安全なモデルファイル形式（.safetensors） |
