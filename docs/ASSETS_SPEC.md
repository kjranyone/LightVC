# LightVC Visual Assets Specification

## Design Direction

**Kawaii Future Bass** — ネオンパステル、グロー、きらめき、丸み。

**カラーパレット:**
```
BG_DARK     #1C1626   (深紫の背景)
BG_PANEL    #2A2038   (パネル背景)
PINK        #FF82BE   (メインアクセント)
PINK_BRIGHT #FFA0D2   (ハイライト)
LAVENDER    #AA8CFF   (セカンドアクセント)
CYAN        #78E6FF   (グロー効果)
MINT        #82FFC8   (正常状態)
YELLOW      #FFDC82   (警告/バイパス)
```

**デザイン原則:**
- 丸み（角丸最小10px、ボタンはピル型）
- グロー（アクティブ要素に柔らかい外光）
- パーティクル（星形、ドット装飾）
- 太めの丸ゴシック体（ポップで可愛い）

## 必要な素材リスト

### 1. アプリアイコン / プラグインアイコン

| ファイル | サイズ | 説明 |
|----------|--------|------|
| `icon_1024.png` | 1024×1024 | マスター（VST3/CLAPバンドル、タスクバー、App Store用） |
| `icon_512.png` | 512×512 | DAWプラグインブラウザ用 |
| `icon_256.png` | 256×256 | Windowsエクスプローラ用 |
| `icon_128.png` | 128×128 | 小サイズ |
| `icon_64.png` | 64×64 | 最小サイズ |
| `icon.ico` | マルチサイズ | Windows用アイコン（16/32/48/64/128/256を内包） |
| `icon.icns` | マルチサイズ | macOS用アイコン |

**デザイン内容:**
- 中央に「✦」（スター型スパンクル）を大きく配置
- スターの色：白〜ピンクのグラデーション
- 背景：深紫（#1C1626）〜ラベンダー（#AA8CFF）の円形グラデーション
- スターの周りに小さなスパンクル（3-5個、シアン/ミント/イエロー）
- 外周に薄いピンクのグローリング
- 全体が丸角正方形のアイコン枠に収まる

**参考イメージ:** Synth V、Serum、Vital のアイコンに近い世代の、ポップでネオン感のあるデザイン

### 2. ロゴ（ヘッダー用）

| ファイル | サイズ | 説明 |
|----------|--------|------|
| `logo_header.png` | 320×80 | アプリ上部バー用（透明背景） |
| `logo_header@2x.png` | 640×160 | Retina用 |

**デザイン内容:**
- テキスト「LightVC」または「✦ LightVC」
- フォント：丸ゴシック、太め（例: Nunito Bold, Quicksand Bold, M PLUS Rounded 1c）
- 「LightVC」部分：白〜シアンのグラデーション
- 「-X」部分：ピンク〜ラベンダーのグラデーション
- 左に小さなスパンクル（✦）
- 背景透過

---

### 3. GUI背景テクスチャ（オプション）

| ファイル | サイズ | 説明 |
|----------|--------|------|
| `bg_texture.png` | 1920×1080 | アプリ全体の背景テクスチャ |
| `bg_texture@2x.png` | 3840×2160 | 4K対応 |

**デザイン内容:**
- 基本色：深紫（#1C1626）
- 左上から右下への微細なグラデーション（#1C1626 → #241A32）
- 全体に散らばる微小なスパンクル（20-30個）
  - サイズ1-3px、色は薄ピンク/シアン/白
  - 透明度10-40%（控えめ、目立ちすぎない）
- 右下に薄いラベンダーのグロー（半径300px、透明度5%）
- ノイズテクスチャ（極薄、2-3%）
- 全体的に「夜空にきらめく星」のような雰囲気

---

### 4. ノブ素材（プラグインエディタ用）

| ファイル | サイズ | 説明 |
|----------|--------|------|
| `knob_64.png` | 64×64 | ノブの静止画（12時方向） |
| `knob_64_frames.png` | 64×768 | 12フレームスプライトシート（0°〜330°、30°刻み） |

**デザイン内容:**
- 丸型ノブ、直径48px程度（余白含め64px）
- ノブ本体：深紫（#2A2038）〜ラベンダーの円形グラデーション
- 外周リング：シアン（#78E6FF）、2px幅
- 現在位置インジケータ：ピンク（#FF82BE）、ノブ中心から外周への線
- 12時位置に小さなスパンクルマーカー
- ノブ中心に極小のピンクドット
- 背景透過

**スプライトシート:**
- 1行64px × 12行 = 768px縦
- 上から0°(7時), 30°, 60°, ..., 330°(5時)
- インジケータ線が回転する

## 実装への影響

アセット追加後、以下を実装します:

1. **アイコン**: `eframe::ViewportBuilder::with_icon()` でタスクバーアイコン設定
2. **背景テクスチャ**: `egui::TextureHandle` で背景画像をロード・描画
3. **ノブ**: eguiのカスタムwidgetでスプライトシートからフレーム選択表示
4. **ロゴ**: ヘッダー部分に `egui::Image` で配置
5. **スプラッシュ**: 起動時に別ウィンドウで0.5秒表示（オプション）

## 注意事項

- **ライセンス**: 全素材はMITライセンス（プロジェクト全体と整合）
- **フォント**: ロゴに使うフォントは商用利用可能なもの（Nunito, Quicksand等はOFL）
- **配色**: 上記カラーパレットを厳守（テーマコード `theme.rs` と一致させる）
- **解像度**: 全PNGは透過背景（alpha channel付き）
