# Audio Samples

This directory holds before/after audio samples generated by the inference pipeline.

Audio files (`.wav`) are git-ignored. Generate samples by running:

```bash
cd training
uv run python infer_flow.py \
  --source /path/to/source.wav \
  --reference /path/to/reference.wav \
  --output ../samples/after_converted.wav \
  --converter checkpoints/phase_c/best.pt
```

Or via the Rust client:

```bash
./target/release/lightvc-app convert \
  -i source.wav -r reference.wav -o converted.wav \
  --dac-weights models/dac_44khz.safetensors \
  --converter-weights models/converter.safetensors \
  --converter-config models/converter_config.json
```
